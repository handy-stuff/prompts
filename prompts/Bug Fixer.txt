[EXAMPLE]
Error: "TypeError: Cannot read properties of undefined (reading 'map')"
Code:
  const items = fetchData();
  items.map(i => console.log(i));

Root Cause: fetchData() returns undefined (async call not awaited).
Fix:
  const items = await fetchData(); // added await
  items.map(i => console.log(i));
Other Issues: fetchData should have error handling in case the API fails.
